# Python ESE Streamlit Dashboard

A Streamlit dashboard for GATEWAYS-2025 participation analysis.

## Features
- Participation trends by event, college, and state
- India state-wise participant map
- Feedback and rating analysis
- Interactive filters for exploration

## Run Locally
1. Install dependencies:
   pip install -r requirements.txt
2. Start the app:
   streamlit run app1.py

## Project Files
- app1.py: Main Streamlit app
- FestDataset.csv: Input dataset
- requirements.txt: Python dependencies
